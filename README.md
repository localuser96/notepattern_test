# PULSE LAB

DJMAX 스타일의 건반형 리듬 패턴을 직접 만들고 바로 테스트하는 로컬 웹 도구입니다.
현재 버전은 별도 설치 패키지나 서버 계정 없이 동작합니다.

## 실행 방법

1. 이 폴더에서 터미널을 엽니다.
2. Windows PowerShell에서는 `npm.cmd start`를 입력합니다.
3. 브라우저에서 `http://localhost:4173`을 엽니다.

다른 터미널에서는 일반적인 `npm start` 명령도 사용할 수 있습니다. PowerShell 실행 정책 때문에
`npm.ps1` 서명 오류가 보이는 경우에는 위의 `npm.cmd start`를 사용하면 됩니다.

Node.js가 설치되어 있지 않다면 [Node.js LTS](https://nodejs.org/)를 먼저 설치해야 합니다.

## 사용 방법

- `BPM`, `키 수`, `마디 수`, `노트 속도`를 설정합니다.
- 단노트 도구는 격자를 한 번 클릭해 노트를 배치합니다.
- 롱노트 도구는 같은 레인에서 시작 위치와 끝 위치를 차례로 클릭합니다.
- `테스트 시작`을 누르면 4박 카운트인 후 플레이가 시작됩니다.
- 키 배치는 에디터 상단과 플레이필드 하단에 표시됩니다.

| 키 수 | 기본 키 배치 |
| --- | --- |
| 4 KEY | `D F J K` |
| 5 KEY | `S D F J K` |
| 6 KEY | `S D F J K L` |
| 8 KEY | `A S D F J K L ;` |

편집 내용은 브라우저의 로컬 저장소에 자동 저장됩니다. 다른 브라우저로 옮길 때는
`JSON 내보내기`와 `JSON 불러오기`를 사용하세요.

## 테스트

```powershell
npm.cmd test
```

판정 범위는 현재 `PERFECT ±45ms`, `GREAT ±90ms`, `GOOD ±140ms`입니다.

## 현재 범위

- 4/5/6/8키 지원
- 16분음표 격자 편집
- 단노트 및 롱노트
- BPM, 마디 수, 스크롤 속도 조절
- PERFECT/GREAT/GOOD 판정 범위와 입력 오프셋 조절
- 키 배치, 카운트인, 메트로놈 및 키 입력 효과음 설정
- 키 입력 판정, 점수, 콤보, 정확도, 결과 등급
- 테스트 완료 후 원본 노트와 실제 입력 위치 비교
- 원본 노트와 입력 노트가 함께 낙하하는 결과 리플레이
- 결과 리플레이와 전체 패턴 비교 화면 전환
- 플레이 시작 시 중앙 집중 모드
- 단일 GPU 합성 레이어 기반의 고프레임 노트 이동
- 마지막 노트의 판정 가능 시간이 끝나면 즉시 테스트 종료
- 자동 저장, 실행 취소, JSON 입출력

음원 업로드, 오디오 파형 동기화, 키 설정 변경, 패턴 공유 기능은 다음 단계에서 확장하기 좋은
영역으로 남겨두었습니다.
